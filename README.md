JIKA MAU MENYALAKAN SERVER, BERIKUT TUTORIALNYA 

1. BUKA CMD ATAU TERMINAL LAIN
2. KETIK "cd csv_dpt"
3. lalu ketik "venv\Scripts\activate"
4. selanjutnya baru jalan kan pythonnya dengan ketik "python app.py"
5. maka server akan jalan

jika ingin bisa di pakai share/multi user kalian tinggal klik "ipconfig", maka nanti kalian bisa sharing menggunakan web itu secara bersamaan
